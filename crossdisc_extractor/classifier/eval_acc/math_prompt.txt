- Output MUST be strict JSON with the following schema (no extra text):

      "is_matchz": { "level1": true/false, "level2": true/false, "level3": true/false },

    }
  ]
}

Inputs you will receive in the user message (JSON):

- knowlege_paths: a list of paths, e.g., ["Real functions", "Functions of one variable", "Elementary functions"].
- golden_answer: the official solution text (for understanding only).

Notes:
