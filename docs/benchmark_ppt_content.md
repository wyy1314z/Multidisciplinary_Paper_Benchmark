## Slide 2: 问题与动机

**标题**: AI for Science 基准的关键缺口

**内容（三列对比）**:

| 缺口 | 现状 | 我们的方案 |
|------|------|-----------|
| 评估深度 | 聚焦代码执行 (MLAgentBench, ScienceAgentBench) | 评估跨学科推理与假设合成 |
| 指标结构 | 文本相似度 / LLM 自评 (IdeaBench) | 图驱动指标 (bridging, path consistency) |
| 评估客观性 | 缺少 Ground Truth | 以真实文献的跨学科路径为参照 |
| 推理可追溯性 | 黑盒生成 | 结构化推理链 + 知识路径 |

**讲稿**:
> 现有 AI for Science 基准关注"LLM 能不能跑代码"或"能不能生成一段假设文本"。但科学发现的核心能力是跨学科构想——系统性地桥接不同学科知识来提出新假设。目前没有基准评估这一能力。CrossDisc 填补了这一关键缺口。

## Slide 3: 方法概览 — 三阶段管线

**标题**: CrossDisc Pipeline: 从论文到结构化假设

**内容（流程图）**:

```
┌──────────┐    ┌──────────────────┐    ┌──────────────────┐    ┌──────────────────┐
│  论文输入  │ →  │ Stage 1           │ →  │ Stage 2           │ →  │ Stage 3           │
│ title     │    │ 学科分类           │    │ 三级查询生成       │    │ 三级假设生成       │
│ abstract  │    │ 概念抽取           │    │ L1: 宏观瓶颈       │    │ L1: 宏观假设       │
│           │    │ 跨学科关系抽取     │    │ L2: 学科问题       │    │ L2: 学科假设       │
│           │    │                    │    │ L3: 操作问题       │    │ L3: 深层假设       │
└──────────┘    └──────────────────┘    └──────────────────┘    └──────────────────┘
```

**讲稿**:
> 管线分三阶段：首先从论文抽取学科概念和跨学科关系；然后生成三个层级的研究查询；最后基于结构化知识生成三级假设。每条假设由推理链或知识路径组成，实体锚定在真实科学概念上。

## Slide 5: 核心创新 2 — 推理链与知识路径

**标题**: 创新 2: 可追溯的推理过程

**内容（左右对比）**:

**左侧 — P4: 推理链 (Reasoning Chain)**
```
L1 hypothesis: Chronic ozone exposure induces
sex-specific lipid peroxidation gradients...

reasoning:
  1. Premise: Ozone alters lipid saturation
     (known from fatty acid studies)
  2. Mechanism: Phospholipase A2 activation
     by oxidative stress (biochemistry)
  3. Prediction: ML segmentation identifies
     alveolar regions with elevated ratios

key_concepts: phospholipase A2, lipid
              peroxidation, mast cell...
disciplines_bridged: 生物学, 生物化学
```

**右侧 — P5: 3-Step 知识路径 (Knowledge Path)**
```
── L1 假设路径 1 ──
  [吸入性毒物] ──引发──→ [空间差异]
  [空间差异] ──对应──→ [脂质种类]
  [脂质种类] ──需解析──→ [生物化学]

  Summary: 吸入性毒物通过引发空间差异性
  脂质分布，需生物化学解析代谢通路
```

**数据对比**:
| 特征 | Baseline (最佳) | P4 | P5 |
|------|----------------|----|----|
| 推理链 | 0/36 | **6/6** | — |
| 学科标注 | 0/36 | **6/6** | — |
| 知识路径 | 0/36 | — | **6/6** |

**讲稿**:
> P4 和 P5 提供了两种互补的可追溯性。P4 是推理链——premise, mechanism, prediction 三步推理，并显式标注关键概念和桥接学科。P5 是知识路径——概念到概念的三步链式连接，每步有严格的实体约束。这两种结构化表示在所有 baseline 中完全不存在。

## Slide 12: 评估框架贡献

**标题**: 超越 ROUGE: 图驱动评估体系

**内容（分层展示）**:

**传统评估（文本层面）**:
- ROUGE / BERTScore → 只看文本重叠

**CrossDisc 评估（多层面）**:
- **客观图指标**（无需 LLM）:
  - Bridging Score: 量化跨学科跳跃距离
  - Path Consistency: 推理链精度/召回
  - Chain Coherence: 逐跳语义连贯性
  - Atypical Combination Index: 学科组合罕见度
- **LLM 评估**（5 维度）:
  - 创新性 / 具体性 / 可行性 / 相关性 / 跨学科性
- **Ground Truth**: 以真实科学文献的跨学科路径为参照

**讲稿**:
> 评估框架本身也是重要贡献。ROUGE 无法衡量推理质量，LLM 自评存在循环论证。我们提出了基于知识图谱拓扑的客观指标：bridging score 量化跨学科跳跃距离，path consistency 衡量推理链质量。以真实文献路径为 ground truth，避免了主观性问题。

## Slide 13: 总结与贡献

**标题**: CrossDisc 的核心贡献

**内容**:

**贡献 1 — 首个跨学科假设生成基准**
- 评估 LLM 跨学科科学推理能力，而非代码执行能力

**贡献 2 — 层级化假设结构 (L1/L2/L3)**
- Baseline: 0/36 → CrossDisc: 24/24 (100%)

**贡献 3 — 可追溯的推理过程**
- P4 推理链: 0/36 → 6/6 | P5 知识路径: 0/36 → 6/6

**贡献 4 — P1-P5 消融框架**
- 量化验证: 结构化知识引导使假设数从 1.0→7.0 条, 层级从无→L1/L2/L3

**贡献 5 — 图驱动评估体系**
- 超越文本相似度, 10+ 客观图指标 + Ground Truth

**One-liner**:
> CrossDisc 不仅让 LLM 生成更好的假设，更展示了假设是**如何**跨学科推导出来的。

**讲稿**:
> 总结五大贡献。核心信息只有一句话：CrossDisc 不仅让 LLM 生成更好的假设，更展示了假设是如何跨学科推导出来的。这是从"写文本"到"做推理"的范式转变。
